Projects

EDA on KUCBT Score

An exploratory data analysis of KUCBT (Kathmandu University Computer-Based Test) scores across engineering departments using data extracted from Kathmandu University's official admissions website. The project analyzes admission trends to answer questions such as which engineering department is the most competitive, the cutoff scores for each department, and the gender ratio across departments. It uses Python, Pandas, NumPy, Matplotlib, and Tabula for PDF extraction, data cleaning, analysis, and visualization.

KUOSC Newsletter (Open Source)

A small TypeScript service that generates a GitHub activity digest (recent repos + commits + issues + PRs) and sends it as an HTML email on a schedule. It uses the GitHub GraphQL API to fetch activity, renders a digest email template, and sends mails.

typescript GraphQL
altEco (Group Project)

altEco is a web-based recommendation system that offers a seamless, intelligent and eco-conscious experience and helps users discover eco-friendly products. It uses TF-IDF vectorization and cosine similarity to match product tags and provide smart suggestions.

javascript postgres express nodeJS
KU Connect (Group Project)

KUConnect is a platform designed to connect university students, facilitating collaboration, networking, and academic discussions. Built using the MERN (MongoDB, Express, React, Node.js) stack, KUConnect provides a seamless experience for students to engage with peers, join study groups, share resources, and stay updated on university events.
